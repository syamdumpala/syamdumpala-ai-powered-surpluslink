# SurplusLink AI

SurplusLink is an AI-powered, real-time food redistribution platform designed to reduce food waste by intelligently connecting surplus food sources with those in need.

The system enables restaurants to share excess food, matches it with NGOs based on demand and priority, and allows volunteers to coordinate delivery in real time. It replaces manual coordination with a structured, data-driven approach.

## Tech Stack

Backend: FastAPI  
Database: PostgreSQL  
Frontend: React with TypeScript  
Styling: Tailwind CSS  
Real-time: WebSockets  
Version Control: Git and GitHub  

## Status

Under active development with core backend and frontend systems in place.

## Vision

To build a scalable platform that minimizes food waste and maximizes impact through intelligent redistribution.
